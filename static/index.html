139Pan Manager

移动云盘分享链接挂载管理与 STRM 生成工具箱

本工具用于将中国移动云盘（139 云盘）分享链接挂载到 AList 中，并自动为视频文件生成 STRM 文件（Emby / Jellyfin / Plex / 飞牛影视可直接读取的快捷方式）。

核心功能
  • • 批量识别分享链接（URL / 纯ID / 名称:ID）
  • • 多令牌管理，一键切换
  • • 自动创建 AList 存储并定时刷新
  • • 递归生成 STRM 文件，支持增量更新
  • • 批量修改 WebDAV 基准 URL
  • • 空文件夹智能清理（两种规则）
技术栈
  • • Python 3.11 + FastAPI + Uvicorn
  • • SQLite3（数据持久化）
  • • Alpine.js + TailwindCSS（前端）
  • • APScheduler（定时任务）
  • • 支持飞牛OS 原生打包（.fpk）
开源协议：MIT
